# 🛒 FreshStore - Modern Grocery Management App

FreshStore ek high-performance grocery listing aur cart management application hai. Isse modern web standards (2026) ko dhyan mein rakh kar banaya gaya hai, jisme focus **State Management** aur **User Experience** par hai.

## ✨ Features

-  **Dynamic Cart:** Items add/remove karne par price turant update hoti hai.
-  **Undo Last Action:** Galti se delete ya add hone par pichle step par wapas jayein.
-  **Discount & Promo System:** - ₹500 se upar ki shopping par automatic 5% threshold discount.
  - Promo codes (e.g., `SAVE10`, `FRESH20`) ke liye manual input support.
-  **Advanced Filtering:** Search bar, category-wise filtering, aur price sorting.
-  **Pagination:** TanStack Query ke saath optimized client-side pagination.
-  **Wishlist System:** Items ko like/unlike karne ki suvidha.
-  **Local Storage Persistence:** Page refresh karne par bhi aapka cart safe rehta hai (via Zustand).
-  **Modern UI:** Glassmorphism, Framer Motion animations, aur Lucide icons ka use.

##  Tech Stack

- **Framework:** [Next.js 15 (App Router)](https://nextjs.org/)
- **Data Fetching:** [TanStack Query v5](https://tanstack.com/query/latest)
- **State Management:** [Zustand](https://zustand-demo.pmnd.rs/) (with Persist Middleware)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)
